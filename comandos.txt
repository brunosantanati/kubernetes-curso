- Subir o container baseado no arquivo docker-compose.yaml:

sudo docker-compose up -d
sudo docker-compose up --force-recreate db

- Ver containers rodando:

sudo docker ps

- Matar container

sudo docker kill projeto_web_1

- Criar imagem customizada:

sudo docker build -t "aplicacao-loja:v1" .

- Criar tabelas no container do banco:

sudo docker exec -it projeto_db1 sh
mysql -u root
use loja
//comandos sql aqui

- Endereço da aplicação

http://localhost:8080
